Query parameters

Specific tables: ?table=

https://www.wikitable2json.com/api/Arhaan_Khan?table=0&table=1

Non-English page: ?lang=

https://www.wikitable2json.com/api/Liste_der_Kulturdenkmale_in_Dedeleben?lang=de

The default language for a page is English (en.wikipedia.com). To get a page that is not in the English subdomain, use ?lang= for the correct language subdomain.
